THE DAY BEFORE
  • Upload player photos — drag each photo into uploads/players/<Player Name>/
  • Producer → Rosters tab → assign photos to players (📷 button next to each player)
  • Verify rosters are complete — correct numbers, names, spelling
  • Double-click Start NSMT.command at home — confirm server starts and producer opens
  • Open http://mbp.local:8000/nsmt_hoopfest_overlay.html — confirm scorebug loads
  • Close server when done testing
  • Charge MBP fully
  • Charge all iPad/tablet devices
1
VENUE ARRIVAL — SERVER & NETWORK

Network

  1. Connect MBP to venue WiFi — or start a personal hotspot on your phone (more reliable)
  2. Open Terminal and run: ipconfig getifaddr en0
  3. Write down the IP address — needed if MBP.local doesn't load on iPads

Start the Server

  1. Open the Livestream folder
  2. Double-click Start NSMT.command
  3. Terminal opens — do not close it for the entire event
  4. The producer control panel opens automatically in the browser
  5. Check the top-right of the producer — confirm the green WS badge is showing

OBS — BROWSER SOURCES
Every Browser Source: Width 1920, Height 1080, transparent background ON, "Shutdown when not visible" OFF, "Refresh on scene activation" OFF.
  1. In your main game scene, add a Browser Source:
    URL: http://mbp.local:8000/nsmt_hoopfest_overlay.html
  2. Confirm the scorebug appears in the OBS preview
  3. If 3-Point Contest: add a second Browser Source in your 3PT scene:
    URL: http://mbp.local:8000/3pt-overlay.html
  4. If Dunk Contest: add a third Browser Source in your Dunk scene:
    URL: http://mbp.local:8000/dunk-overlay.html

GAME SETUP WIZARD (Producer — Tab 1)
Run this wizard for every game, including between games on a multi-game day.

Step 1: Game Info

Event name · Venue · # of Periods (4 for quarters, 2 for halves) · Period length (8 min default) · Sponsor label

Step 2: Away Team

Team name · Code (3–4 chars for scorebug, e.g. "WST") · Logo · Colors · Roster assignment

Step 3: Home Team

Same fields as Away.

Step 4: Review & Start

  1. Review both teams in the summary
  2. Tap ▶ Start Broadcast
  3. ✓ Scorebug in OBS updates with team names, logos, colors, 0–0

HALFTIME
  1. iPad Game tab → tap End Period ▸ (advances from Q2 → H2)
  2. MBP Producer → Broadcast tab → tap Halftime
    → HALFTIME graphic fires with both scores
  3. During the break: update roster if needed — no need to reset scores

END OF GAME
  1. MBP Producer → Broadcast tab → tap End of Game
    → FINAL graphic fires with the final score
  2. iPad → Game tab → tap ↺ Reset Game → confirm
    → Clears scores to 0–0, resets clock, clears stats, keeps rosters for next game

3-POINT CONTEST
Switch to your 3PT scene in OBS first.

Round 1 — Two courts simultaneously

Primary device (Court 1):

  1. iPad → 🏀 3PT tab → tap Court 1 → tap Round 1
  2. Type shooter's name (or select from dropdown)
  3. Tap the ★ Money Rack button (1–5) — shooter picks which rack is 2 pts
  4. Tap ▶ Show to display the overlay
  5. As each ball is thrown, tap the ball button in that rack row (made = filled, missed = empty)
  6. When shooter finishes: tap Next Shooter ▶ → balls clear, name field resets

Second device (Court 2): runs the same workflow independently. Both courts appear side-by-side on the overlay.

Finals — Single shooter

  1. Primary device → 3PT tab → tap Finals (overlay switches to single centered panel)
  2. Enter finalist's name, select money rack, tap Show
  3. Same ball-tapping workflow as Round 1
  4. Tap Next Shooter ▶ to advance to the next finalist
12
DUNK CONTEST
Switch to your Dunk scene in OBS first.

Head Table — iPad Dunk Tab

  1. Dropdown: select dunker's name (school + Instagram auto-fill on overlay)
  2. Optional: paste a photo URL
  3. Set Round and Attempt numbers (−/+ buttons)
  4. Tap ▶ Show to display the overlay
  5. Watch judge cards — they turn blue as each judge submits (score stays hidden)
  6. Reveal scores: tap ▶ Reveal one judge at a time for drama — or Reveal All for instant
  7. After dunk: tap ↺ Reset Dunk (keeps dunker, clears scores for next attempt)
  8. New dunker: tap New Dunker

Judges — Each on their own device

  1. Open your judge URL (see Section 4 above, or the Dunk tab on the primary iPad)
  2. See dunker name, round, attempt (read-only)
  3. Tap a score: 1 through 10
  4. Tap ✓ Submit Score — score locks, head table card turns blue
  5. Page auto-resets after head table taps Reset Dunk
If a judge refreshes by accident: just reopen the URL and resubmit — it replaces the previous score.
