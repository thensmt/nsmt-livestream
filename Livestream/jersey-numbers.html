NSMT Jersey Numbers

Hoopfest 2026 — Enter jersey numbers before game time

How to use

  1. Enter jersey numbers for each player above.
  2. Click Save to iPad — numbers are saved to this browser's localStorage.
  3. Open the iPad Control Panel from the same device/browser. When you activate a preset (Nova 24 or PvP), jersey numbers will load automatically.
  4. If using a different browser/device for the iPad panel, click Copy JSON, then paste the JSON into that browser's console: localStorage.setItem('nsmt_jersey_numbers', '...')
  5. Numbers persist until you clear them. You can update anytime — just change values and click Save again.
