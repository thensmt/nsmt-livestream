📅 Game Day Timeline
48-72 Hours Before
  • Confirm game: teams, venue, date/time
  • Open Roster Manager → verify both teams saved
  • Add new/missing players to rosters
  • Charge iPad fully
  • Confirm app is on home screen (Safari → Share → Add to Home Screen)
Arrive at Venue (45 Min Before)
  • Open NSMT Stats → New Game → your name
  • Select Stat-Keeper Mode
  • Enter event, venue, date, time
  • Load Away roster → verify players
  • Load Home roster → verify players
  • Add missing players manually if needed
  • Save Rosters → Start Game
5 Min Before Tip
  • Identify starting lineups (both teams)
  • Find courtside seat — clear view of both benches
  • Set clock: tap 🕐 → enter period length (8:00)
  • Quick test: tap a player → 2PT → tap Back
After Final Buzzer
  • Tap End → confirm score → End Game
  • Review box score totals quickly
  • Tap 🖨️ Print / Save PDF → save or share
  • Tap 📦 Export .nsmtgame → send to NSMT HQ
  • Screenshot final score for social

⚡ Quick Reference
📍 Shot Location
🟥 PaintInside the lane, within ~8 ft of basket
🟨 Mid RangeBetween paint and 3PT line
🟦 3-PointBehind the arc (auto-set for 3PT shots)
🔄 Play Context
⛹️ Half CourtDefault — normal set half-court play
💨 Fast BreakScore off steal or quick outlet
🔄 2nd ChanceScore after offensive rebound
💥 Off TurnoverScore after opponent TO (not fast break)
🎛️ Bottom Bar Controls
↩ UndoRemove last logged event
🕐 ClockSet current game time (MM:SS)
⏱ PeriodEnd period, advance, or add OT
📋 ReportPreview stats (doesn't end game)
EndFinalize game, generate official report

🔴 Emergency Procedures
App crashes: Re-open → tap the recent game on home screen. All events auto-save — no data lost.
Wrong stat logged: Tap ↩ Undo immediately and re-log with correct player/action.
No Wi-Fi at venue: Not an issue. App is 100% offline. Just track and export after.
iPad dies mid-game: Charge and reopen. Game resumes from last auto-save in archive.
